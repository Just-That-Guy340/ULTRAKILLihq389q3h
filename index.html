ULTRAKILL

Source Code Repository

Welcome to the ULTRAKILL source code repository. This is a collection of C# scripts and assets from the fast-paced retro FPS game ULTRAKILL.

⚠️ Note: This repository contains the source code only. To play ULTRAKILL, visit the official game page.

This repository includes:

  • Game mechanics and systems (weapons, enemies, physics)
  • UI and menu systems
  • Audio and visual effects
  • Level design and gameplay systems
  • Custom utilities and extensions
